Everything for Github can be done in VSCode

  1. Initialize a folder to reflect your repository (probably have already done this).
    • Open a folder and/or a file
    • Go to the Source Control trident on the left and click into it
    • Click Initialize Repository

  2. A trident 'main' 'asterik' will now be on the bottom left.

  3. Your file (in this case, index.html) will have a green U to the right of it.
    • A green U signals an "Untracked" file
    • Untracked files are file changes that have not been added to the repository

  4. Track this file by covering by the U and pressing the + button.
    • Pressing + stages the code

  5. Staged files now have an A next to it
    • The A means the file has been staged but that's the only thing that plus does
    • Don't click anything yet: at this point, the action is only being described

  6. Next, you'll be committing the staged changes.
